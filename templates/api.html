Image Conversion API

POST /api/convert

Convert Image

Convert images between different formats with optional quality settings.

Parameters

Name Type Description
file File The image file to convert
format String Target format (png, jpg, webp)
quality Number Output quality (1-100)

Example Request

curl -X POST \
  'https://yourdomain.com/api/convert' \
  -F 'file=@image.png' \
  -F 'format=jpg' \
  -F 'quality=85'

YouTube API

POST /api/youtube/info

Get Video Information

Retrieve information about a YouTube video including available formats.

Parameters

Name Type Description
url String YouTube video URL

Example Request

curl -X POST \
  'https://yourdomain.com/api/youtube/info' \
  -F 'url=https://youtube.com/watch?v=...'
